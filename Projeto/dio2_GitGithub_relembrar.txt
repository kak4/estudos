- entra na interface do Github

- clicar no "+" e em novo repositorio

  "publico" ou "privado": melhor publico para outras empresas e pessoas verem. Mas voce escolhe
    "add a README file" colocsr um README( é como se fosse a "pagina principal" )
    "add .gitgnore": uma tecnologia que ignora uns arquivos para não atrapalhar
    "chose a license" um conjunto de licensas
    
    "chose a license" e "add .gitgnore"
